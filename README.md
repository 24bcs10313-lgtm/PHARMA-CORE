# PHARMA-CORE 💊

PHARMA-CORE is a full-stack pharmacy web application created to make medicine searching and healthcare product management simple and fast.  
This project was built as a learning-based full stack development project using React, Node.js, Express, and SQLite.

The application provides a clean modern interface where users can search medicines and explore healthcare products with a responsive frontend and database-powered backend.

---

# 🚀 Features

- 🔍 Search medicines instantly
- 💊 Medicine catalog system
- 🔐 Login & Signup pages
- ⚡ Fast backend API
- 🗄️ SQLite database integration
- 📱 Responsive modern UI
- 🧩 Full-stack project structure

# ▶️ How to Run the Project

## Clone the Repository

```bash
git clone https://github.com/24bcs10313-lgtm/PHARMA-CORE.git
```

---

## Start Backend Server

```bash
cd backend
npm install
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```

---

## Start Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
